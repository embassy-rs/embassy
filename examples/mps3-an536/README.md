# Examples for Cortex-R52 on MPS3-AN536 (in QEMU)

Run individual examples with:

```sh
cargo run --bin <crate-name>
```

for example:

```sh
cargo run --bin hello
```

## Checklist before running examples

Before running these examples, you will need:

* To have `qemu-run` on your PATH (try `cargo install qemu-run`)
* To have `qemu-system-arm`, version 9 or higher, in your PATH
